Shape importer is used to import shape files in the Ktunaxa database

HOW-TO CONFIGURE DATABASE USER
- unzip ktunaxa.jar
- modify parameters in org/ktunaxa/referral/shapereader/spring-hibernate.xml

HOW-TO RUN ON WINDOWS 
- open DOS prompt 
- type shapeImport

HOW-TO RUN ON UNIX
- open shell
- chmod +x shapeImport
- ./shapeImport
